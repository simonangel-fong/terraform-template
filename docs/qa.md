# QA Commands

Commands to run from a module directory (`<cloud>/<name>/`) to ensure code quality.

## Validate

```sh
# Format
terraform fmt -recursive -diff

# Init without backend (downloads providers only)
terraform init -backend=false

# Catch syntax / type errors
terraform validate

# Static analysis
tflint --recursive
checkov -d . --framework terraform
```

## Test

```sh
terraform test -verbose
```

## Generate README

The module's `README.md` is generated by `terraform-docs` from `.terraform-docs.yml`.

Create `README.md` containing the inject markers, then run terraform-docs:

```markdown
<!-- BEGIN_TF_DOCS -->
<!-- END_TF_DOCS -->
```

```sh
terraform-docs .
```

Re-run after any change to variables or outputs.
